En este párrafo probaremos todas las etiquetas vistas hasta ahora:
Con strong podemos poner un texto o palabra en negrita También tenemos la etiqueta em para poner un texto en cursiva o la etiqueta span que no hace nada sobre un texto pero permite añadir identificadores para posteriormente en css dar un estilo mas rapido
Tambien tenemos la etiqueta i, que al igual que em pone un texto en cursiva
Por último veremos la etiqueta blockquote que permite poner un texto tipo cita como el siguiente:
